Instrucciones rápidas:
1) Copia la carpeta 'mi_panel' dentro de C:\xampp\htdocs\
2) En XAMPP Control Panel inicia Apache y MySQL.
3) Abre en tu navegador: http://localhost/mi_panel/create_admin.php
   - Verás un mensaje que indica que el admin fue creado/actualizado.
   - Importante: ELIMINA create_admin.php después de usarlo (por seguridad).
4) Accede al login: http://localhost/mi_panel/index.php
   Usuario: admin
   Contraseña: Camilo2012$
5) Para cambiar credenciales, entra a la base de datos en phpMyAdmin o agrega funciones de gestión de usuarios.
